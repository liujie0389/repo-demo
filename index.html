在 github 建立项目演示页面

  1. 在本地新建 repo-demo 项目
  2. 初始化本地项目：git init
  3. 将所有文件加入项目：git add .
  4. 提交项目更新：git commit -m 'demo page for repo-demo'
  5. 为项目新建一个 gh-pages 分支：git branch gh-pages
  6. 切换到 gh-pages 分支：git checkout gh-pages
  7. 删除本地 master 分支（可选）：git branch -D master
  8. 在 github 新建 repo-demo 项目
  9. 添加远端路径：git remote add origin git@github.com:forcefront/repo-demo.git
  10. 推送本地的 gh-pages 分支至远端：git push origin gh-pages
  11. 查看示例项目：checkout http://github.com/forcefront/repo-demo
  12. 访问项目演示地址：and visit http://forcefront.github.com/repo-demo
